MetaABC_Mod

homepage

MetaABC is a pipeline that integrates advanced binning tools coupled with data filters and normalizations for improving metagenomic analysis. It incorporates two means for removing artifacts, five methods for taxonomic binning, an approach to re-analyze unassigned reads, and an option to control sampling biases via genome-length normalization. It can handle sequencing data generated using different technologies. In a single platform, MetaABC furnishes various analyzing routes that allow users to reach an optimized outcome. Further, different output visualization formats (includingtables, pie charts, and bar charts of abundance profiles) can be generated per request, and a hierarchical clustering program is included for comparative analysis. To facilitate the computation processfor larger datasets, a stand-alone version is also available.
